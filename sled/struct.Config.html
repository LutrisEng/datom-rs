Struct sled::Config

source · 
pub struct Config(/* private fields */);
Expand description

Top-level configuration for the system.

Examples

let _config = sled::Config::default()
    .path("/path/to/data".to_owned())
    .cache_capacity(10_000_000_000)
    .flush_every_ms(Some(1000));
